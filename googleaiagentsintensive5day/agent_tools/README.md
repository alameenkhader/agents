## Setup

### Setup and activate venv
python3 -m venv venv # To setup venv, run this only once to setup
source venv/bin/activate

### Install dependencies
python3 -m pip install google-adk

### ENVS

Create .env file.
Make sure to set up env GOOGLE_API_KEY in the .env file
If you do not have one already go to https://aistudio.google.com/ to create one.

## How to run the program

python3 main.py
